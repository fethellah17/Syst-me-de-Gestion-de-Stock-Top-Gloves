## 🎯 Déploiement en 3 Étapes

### Méthode 1 : GitHub + Vercel (Le Plus Simple)

```bash
# 1. Commit et push
git add .
git commit -m "Ready for deployment"
git push

# 2. Aller sur https://vercel.com
# 3. Import Project → Sélectionner votre repo → Deploy
```

### Méthode 2 : CLI Vercel

```bash
# 1. Installer (si pas déjà fait)
npm install -g vercel

# 2. Se connecter
vercel login

# 3. Déployer
vercel --prod
```
